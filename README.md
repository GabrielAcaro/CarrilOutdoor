# Comparador CSV Productos

Programa en Java para comparar productos entre distintos archivos CSV, como ERP, Prestashop y proveedores.

## Objetivo

La herramienta permite:

- Cargar archivos CSV.
- Mostrar productos de forma ordenada.
- Buscar productos por nombre, referencia, EAN, talla o color.
- Detectar posibles coincidencias entre productos.
- Confirmar coincidencias manualmente.
- Generar un código común para productos equivalentes.
- Exportar resultados revisados.
